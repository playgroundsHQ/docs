# Stargate

**Stargate** — це публічна колекція шаблонів додатків для миттєвого запуску. Вона надає курований каталог стеків розробки, які ви можете розгорнути одним кліком.

## Огляд

Stargate показує всі публічно доступні [шаблони](/launch/templates) — попередньо налаштовані блюпринти середовищ, створені командою fibe.gg та спільнотою. Кожен шаблон включає все необхідне для запуску повного стеку: конфігурацію Docker Compose, класифікацію сервісів та визначення змінних.

## Перегляд шаблонів

Шаблони організовані за **категоріями** (напр., Web Frameworks, Databases, Full-Stack Starters). Ви можете:

- **Переглядати за категорією** — Фільтрувати галерею за технологією або варіантом використання
- **Шукати** — Повнотекстовий пошук за назвами та описами шаблонів
- **Переглядати** — Побачити Docker Compose визначення та змінні шаблону перед запуском

## Запуск з Stargate

1. Знайдіть шаблон через перегляд або пошук
2. Натисніть на шаблон для перегляду деталей
3. Заповніть необхідні змінні (напр., назва проєкту, пароль бази даних)
4. Оберіть [Playroom](/core-concepts/playroom) для розгортання
5. Натисніть **Launch**

Платформа створює [Playspec](/core-concepts/playspec) з шаблону та (за вашим вибором) одразу розгортає [Playground](/core-concepts/playground).

## Створення шаблонів для Stargate

Щоб опублікувати шаблон у Stargate:

1. Створіть шаблон у [My Fleet](/launch/my-fleet)
2. Створіть версію з тілом шаблону
3. Встановіть видимість версії на **Public**

Публічні версії доступні для всіх користувачів через Stargate. Див. [Шаблони](/launch/templates) для повного формату шаблонів та системи змінних.
