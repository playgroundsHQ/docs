# Майстер адаптації

**Майстер адаптації** розроблений для того, щоб допомогти новим гравцям швидко та легко розпочати роботу з fibe.gg. Він проведе вас через основні кроки налаштування вашого першого віддаленого хоста та середовища.

## Потік майстра

Майстер складається з чотирьох основних кроків:

### 1. Привітання та вступ
Короткий огляд можливостей платформи та того, що вам знадобиться для початку (доступ до GitHub та віддалений сервер).

### 2. Підключення Playroom
Налаштування вашого першого віддаленого Docker-хоста. Вам потрібно буде вказати:
- **Адресу хоста** (IP або домен)
- **SSH-користувача**
- **SSH-порт** (за замовчуванням 22)
- **Кореневий домен** для ваших середовищ

Майстер автоматично згенерує SSH-ключ для вас та надасть інструкції щодо його додавання до `authorized_keys` вашого сервера.

### 3. Підключення Playzone
Підключення GitHub-репозиторію для використання як джерела для ваших середовищ. Ви можете обрати з:
- **Публічні репозиторії**: Підключення будь-якого публічного репозиторію за URL-адресою.
- **Приватні репозиторії**: Прив'яжіть свій обліковий запис GitHub для доступу до приватних репозиторіїв.

### 4. Запуск першого Playground
Оберіть шаблон або використайте власний Docker Compose файл для створення вашого першого Playground.

## Пропуск майстра
Досвідчені користувачі можуть пропустити майстер у будь-який час для доступу до повної панелі керування та ручних налаштувань. Ви завжди можете перезапустити майстер з налаштувань профілю за потреби.

:::info
Завершення майстра гарантує, що ваш обліковий запис правильно налаштований для стандартного робочого процесу розробки.
:::
