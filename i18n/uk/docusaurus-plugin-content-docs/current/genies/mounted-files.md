# Змонтовані файли Генія

**Змонтовані файли** дозволяють приєднувати файли безпосередньо до Генія. Ці файли автоматично інжектуються в контейнер Генія за визначеним шляхом кожного разу, коли Геній запускається — у будь-якому Playground.

Це ідеально підходить для:
- Конфігураційних файлів Генія (наприклад, `claude.md`, `.opencode.json`)
- Особистих SSH-ключів або токенів, необхідних Генію
- Спільних секретів або сертифікатів

## Додавання змонтованого файлу

Ви можете додавати змонтовані файли через картку деталей Генія або через API.

### Через інтерфейс

1. Відкрийте картку деталей вашого Генія
2. Прокрутіть до розділу **Змонтовані файли**
3. Натисніть **Додати файл**
4. Завантажте файл та встановіть **шлях монтування** (наприклад, `/root/.config/my-genie/config.json`)

### Через API

```bash
POST /api/genies/:id/mounted_files
Content-Type: multipart/form-data

file=@./claude.md&path=/root/.claude/CLAUDE.md
```

## Керування файлами

- Файли зашифровані в стані спокою в Active Storage
- Змінюйте шлях монтування або замінюйте файл у будь-який час
- Видаліть файл, щоб припинити його монтування без повторного створення Генія

## Обмеження

| Обмеження | Значення |
|-----------|----------|
| Максимум файлів на Генія | 3 |
| Максимальний розмір файлу | 2 МБ |

## MCP-інструмент

Генії можуть керувати своїми змонтованими файлами через MCP:

```
add_agent_mounted_file  — Завантажити новий файл до Генія
remove_agent_mounted_file — Видалити змонтований файл з Генія
```

Це дозволяє AI Геніям самостійно налаштовувати власне середовище, завантажуючи конфігураційні файли програмно.
